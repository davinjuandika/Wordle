# 🟩🟨 Wordle Clone ⬜🟩

A simple, fun clone of the popular word-guessing game Wordle, built entirely in Python. 

### 🎮 How to Play
There are two ways to play this game depending on what you prefer:

**Option 1: Play the Game (No setup required)**
1. Download the `wordle.exe` file directly from this repository.
2. Double-click the file to launch the game and start guessing!

**Option 2: For Developers (Run the code)**
1. Clone or download this repository to your computer.
2. Make sure you have Python installed.
3. Open your command prompt or terminal in the project folder and run:
   ```bash
   python Wordle.py
